## Architecture
RAG is modeled as a **multi-stage observation pipeline** inspired by physical light behavior.

Stages:
1. Ingestion (Observation)
2. Chunking (Diffraction)
3. Embedding (Refraction)
4. Retrieval (Reflection)
5. Reasoning (Interpretation)
6. Evaluation (Noise & Hallucination)
